表示数を増やす

  • 現在の最大列数を表示 
    print(pd.get_option("display.max_columns"))
  • 最大列数を50列にする 
    pd.set_option('display.max_columns', 50)
  • 現在の最大行数を表示 
    print(pd.get_option("display.max_rows"))
  • 最大行数を50行にする 
    pd.set_option('display.max_rows', 50)


DataFrameの作成

  • Numpyのndarrayから作成する方法 
    df = pd.DataFrame(
    {
        'index': np.array(index),
        'mean': np.array(dmean),
        'std': np.array(dstd)
    },
    dtype='float')
    インデックスをindex、平均値をdmean、標準偏差をdstdに格納した場合

  • csvファイルから作成する方法 
    df = pd.read_csv(file_name)


リサンプリング

  • 時間軸のデータを日平均する方法 
    df_new = df.resample('D').mean()
  • 時間軸のデータを日毎に合計する方法 
    df_new = df.resample('D').sum()


データの選択

  • 配列の番号で選択する場合 
    df_new = df.iloc[:, 0]
    dfが２次元配列の場合。axis=1の0番目の要素を選択

  • 配列のラベルで選択する場合 
    df_new = df.loc[:, "lab"]
    dfが２次元配列の場合。labという名前の列を選択

  • 条件により選択（Aが正の場合） 
    df_new = df[A >= 0]
    Aはdfと同じサイズで条件を表す配列



データの置き換え

  • 別のデータから置き換え 
    df.iloc[:, 0] = np.array(data)
    dfが２次元配列の場合。axis=1の0番目の要素を選択し、dataで置き換える

  • 条件により選択（負の場合にNaN） 
    df[df < 0] = np.nan

  • 条件により選択（qaが50%未満の場合にNaN） 
    df[qa < 50] = np.nan
    qaはdfと同じサイズの配列
