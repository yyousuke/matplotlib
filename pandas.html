DataFrameの作成

  • Numpyのndarrayから作成する方法 
    df = pd.DataFrame(
    {
        'index': np.array(index),
        'mean': np.array(dmean),
        'std': np.array(dstd)
    },
    dtype='float')
    インデックスをindex、平均値をdmean、標準偏差をdstdに格納した場合

  • csvファイルから作成する方法 
    df = pd.read_csv(file_name)


リサンプリング

  • 時間軸のデータを日平均する方法 
    df_new = df.resample('D').mean()
  • 時間軸のデータを日毎に合計する方法 
    df_new = df.resample('D').sum()


データの選択

  • 配列の番号で選択する場合 
    df_new = df.iloc[0, :]
    dfが２次元配列の場合。0番目の要素を選択
  • 配列のラベルで選択する場合 
    df_new = df.loc["lab", :]
    dfが２次元配列の場合。labという名前のものを選択
  • 条件により選択（Aが正の場合） 
    df_new = df[A >= 0]
    Aはdfと同じサイズで条件を表す配列
